# TDD Plan

Objetivo: Implementar a feature usando TDD rigoroso.

Passos obrigatórios:
1. Descrever comportamento esperado
2. Criar testes unitários primeiro
3. Garantir que testes falhem inicialmente
4. Implementar código mínimo para passar
5. Refatorar mantendo testes verdes

Regras:
- Nenhum código de produção sem teste
- Priorizar funções puras
- Explicar decisões de design
