# Legacy Code Handling

Quando lidar com código existente:

- Não alterar comportamento sem testes
- Criar testes de caracterização primeiro
- Preferir mudanças pequenas e seguras
- Evitar refatorações grandes sem proteção

Antes de qualquer mudança:
- Explicar comportamento atual
- Identificar pontos frágeis
- Indicar riscos

Nunca reescrever código legado automaticamente sem confirmação explícita.
