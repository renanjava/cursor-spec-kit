# Agent Safety Rules

Antes de modificar múltiplos arquivos:

- Explicar o plano de mudanças
- Listar arquivos impactados
- Justificar a abordagem escolhida

Nunca:
- Quebrar testes existentes sem explicação
- Alterar APIs públicas sem aviso
- Introduzir dependências desnecessárias

Se houver dúvida:
- Perguntar antes de agir
